
# Advent2022

<https://adventofcode.com/2022/>

## Language

Rust!

## Usage

### run all days

```bash
cargo run
```

### test example inputs for specific day (where XX is day no)

```bash
cargo test -p day_XX
```

### generate new day (where XX is day no)

```bash
cargo run -p cargo-day -- XX
```

(alternatively)
```bash
cargo install --path cargo-day
cargo-day XX
```
